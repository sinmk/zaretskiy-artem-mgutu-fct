Проект от ФЦТ МГУТУ
Разрабатывается студентами факультета цифровых технологий МГУТУ им. К.Г. Разумовского в рамках подготовки к акселераторам Leader-ID. Презентация проекта: https://pt.2035.university/project/dreema-umnoe-prilozenie-dla-ulucsenie-kacestva-sna

Научная основа
Проект продолжает работу студенческого проекта, проведённого в МГУТУ:
SLEEPWALKER – приложение для тех, кто не высыпается (июнь 2025);
Ключевой метод: анализ данных о сне, поведении и здоровье пользователя, для получения индивидуальных рекомендаций.

Чек-лист первого контрибьютора:
1. Сделайте форк репозитория (кнопка Fork в правом верхнем углу).
2. Клонируйте свой форк локально:
   git clone https://github.com/ВАШ-ЛОГИН/ваше-имя-mgutu-fct.git
3. Перейдите в папку проекта:
   cd ваше-имя-mgutu-fct
4. Установите зависимости:
   pip install -r requirements.txt
5. Запустите тесты:
   python -m pytest tests/
6. Внесите правки → зафиксируйте → отправьте PR.
